# Welcome to TSN-Affinity

Training-Aware Sparse Networks with Affinity Routing for Continual Offline Reinforcement Learning.

## Features

- **Sparse Subnetwork Allocation** via learnable weight scores
- **Affinity Routing** with action, latent, and hybrid modes
- **Frozen Weight Protection** against catastrophic forgetting
- **Decision Transformer Backbone** for offline RL
- **CLI Tools** for synthetic and Atari benchmarks
